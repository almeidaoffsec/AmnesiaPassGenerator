Amnesia Pass Generator

Gerador de senhas determinístico baseado em hash. Insira uma palavra-chave e um salt por serviço — a mesma senha é sempre recriada sem precisar armazená-la em lugar nenhum.

Usar agora Ver no GitHub

Dica: instale o app para uso offline (menu do navegador → Instalar app).

Como funciona

O Amnesia Pass Generator é determinístico: insira sempre a mesma palavra-chave e o mesmo salt e a mesma senha será recriada — sem banco de dados, sem nuvem.

  1. Insira a palavra-chave e, opcionalmente, um salt por serviço.
  2. O sistema gera um hash SHA512 da entrada.
  3. O processo se repete pelo número de iterações definido.
  4. O resultado é truncado para o número de caracteres desejado.
  5. Prefixo e sufixo são adicionados ao resultado final.

Perfis Salvos

Salve suas configurações para reutilizá-las sem redigitar tudo. Cada perfil é cifrado com AES-GCM-256 usando sua própria palavra-chave — nenhum dado sensível fica em texto claro no armazenamento.

  1. Preencha os campos e clique em Salvar Perfil Atual.
  2. Dê um nome ao perfil (ex.: GitHub, Banco).
  3. Para usar, selecione o perfil no dropdown e clique em Carregar.
  4. Insira sua palavra-chave no modal — todos os campos são restaurados.
Exportar e Importar

Use Exportar para baixar um .json com seus perfis cifrados e Importar para restaurá-los em outro dispositivo. O arquivo exportado não contém nenhuma informação em texto claro.

Segurança e Privacidade

Todo o processamento é feito exclusivamente no seu navegador. Nenhuma informação é enviada para servidores externos.

Aviso sobre Iterações

Valores excessivamente altos (ex: milhões) podem causar lentidão ou travamento temporário. Para uso comum, valores de 1 a 1000 são instantâneos.

Dica de Uso

Use prefixo e sufixo para adicionar caracteres especiais e maiúsculas sem alterar o hash base.

resultado 
#MeuPrefixo + [hash gerado] + #MeuSufixo

Use uma seed mestra forte (frase longa) e um salt diferente para cada serviço.
